## 🇭🇰 繁體中文

**開發者：** 蘇小偉 (Su Xiaowei)  
**軟體狀態：** v2.4 Stable (穩定版)

### ✨ 核心功能詳解
* **🌪️ 全自動導航模式：** 專為「伸手黨」設計。點擊後程式會自動掃描左側選單，遍歷包含「第X周」、「第X章」、「Chapter」、「Week」、「Lab」、「作業」等關鍵字的頁面。
* **🎯 智能文件過濾引擎：** 自動過濾 Blackboard 煩人的 `.htm` 網頁導航檔。只提取真正的 PDF、PPTX、Word、ZIP 等教學課件，保持你的資料夾乾乾淨淨。
* **📌 置頂交互 GUI：** 採用置頂 (Always-on-top) 技術，下載小視窗會浮在 Chrome 上方，無需頻繁切換視窗 (Alt+Tab)。
* **🚀 零配置啟動：** 自動偵測系統路徑。檔案統一存放在 `下載/BB_Downloads_by_Su`，並在下載完成後自動彈出資料夾。

### 📖 使用說明
1. **環境準備：** 確保電腦已安裝 **Google Chrome** 瀏覽器。
2. **啟動軟體：** 執行 `BB_Downloader_Pro_v2.4.exe`（初次運行若有防毒警告，請點選「仍要執行」）。
3. **登入系統：** 在自動開啟的瀏覽器中手動完成 Blackboard 登入。
4. **進入課程：** 點進你想要下載的**具體課程頁面**。
5. **開始任務：**
   - **單頁下載：** 若只想下載目前看到的檔案，按「1」或點擊綠色按鈕。
   - **全自動下載：** 點擊藍色按鈕，程式將掃描整個學期的所有章節文件。
6. **結束關閉：** 任務完成後，按「Q」鍵即可乾淨關閉程式與瀏覽器。

<a name="english"></a>
## 🇺🇸 English

**Developer:** Su Xiaowei  
**Status:** v2.4 Stable Release

### ✨ Key Features in Detail
* **🌪️ Full-Auto Navigation:** Designed for efficiency. The tool scans the course sidebar for keywords like "Week", "Chapter", "Lab", "Assignment", and "第X章/周", visiting each page to collect resources.
* **🎯 Smart Content Filter:** Automatically ignores messy `.htm` and `.html` wrapper files. It captures only the essential courseware (PDF, PPTX, DOCX, ZIP, etc.).
* **📌 Floating Controller:** Built with "Always-on-top" logic. The control panel stays visible above your browser for seamless one-click operation.
* **🚀 Zero-Configuration:** Detects the system path automatically. All files are saved to `Downloads/BB_Downloads_by_Su` and the folder opens automatically upon completion.

### 📖 Step-by-Step Guide
1. **Prerequisite:** Ensure **Google Chrome** is installed on your Windows PC.
2. **Launch:** Run `BB_Downloader_Pro_v2.4.exe`.
3. **Authenticate:** Manually log in to the SUSTech Blackboard system in the pop-up browser.
4. **Target Course:** Navigate into the **specific course page** you wish to sync.
5. **Execution:**
   - **Manual Scan:** Click the Green button or press "1" to download files on the current page.
   - **Auto-Sync:** Click the Blue button to scan and download files from all detected chapters/weeks.
6. **Quit:** Press "Q" or click the Red button to close the program and the browser session safely.

## 🇫🇷 Français

**Développeur:** Su Xiaowei  
**Statut:** v2.4 Version Stable

### ✨ Caractéristiques Détaillées
* **🌪️ Mode Navigation Automatique:** Scanne intelligemment le menu latéral pour trouver des mots-clés comme "Semaine", "Chapitre", "Laboratoire", et "Devoirs".
* **🎯 Filtre de Contenu Intelligent:** Ignore les fichiers `.htm` encombrants pour ne conserver que les documents pédagogiques réels (PDF, PPTX, ZIP, etc.).
* **📌 Interface Toujours Visible:** La fenêtre de contrôle reste au premier plan au-dessus de Chrome, facilitant l'interaction sans changer de fenêtre.
* **🚀 Prêt à l'Emploi:** Détecte automatiquement le chemin système. Les fichiers sont enregistrés dans le dossier `Téléchargements` et le dossier s'ouvre automatiquement à la fin.

### 📖 Guide d'Utilisation
1. **Prérequis:** Assurez-vous que **Google Chrome** est installé sur votre PC.
2. **Lancement:** Exécutez `BB_Downloader_Pro_v2.4.exe`.
3. **Connexion:** Connectez-vous manuellement à Blackboard dans le navigateur ouvert par l'outil.
4. **Ciblage:** Entrez dans la **page du cours spécifique** que vous souhaitez télécharger.
5. **Exécution:**
   - **Scan Simple:** Cliquez sur le bouton Vert (ou appuyez sur "1") pour la page actuelle.
   - **Synchro Auto:** Cliquez sur le bouton Bleu pour scanner tous les chapitres de la session.
6. **Quitter:** Appuyez sur "Q" pour fermer proprement l'application et la session Chrome.
