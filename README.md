## 🇭🇰 繁體中文

**開發者：** 蘇小偉 (Su Xiaowei)  
**最新版本：** v2.4 (通用分享版)

### 🌟 簡介
這是一個專為南科大 (SUSTech) 學生設計的自動化工具，旨在解決手動下載 Blackboard 資源的繁瑣過程。

### ✨ 功能特點
* **一鍵全自動：** 自動識別「第X周」、「第X章」、「實驗」、「作業」等目錄。
* **智能文件過濾：** 只下載 PDF, PPTX, DOCX 等課件，自動跳過網頁結構檔案。
* **視窗置頂介面：** 視窗永遠保持在最前方，方便在操作瀏覽器時隨時點擊。
* **全電腦兼容：** 自動下載至「下載」資料夾，無需擔心硬碟代號 (如 E 槽) 問題。

<a name="english"></a>
## 🇺🇸 English

**Developer:** Su Xiaowei  
**Version:** v2.4 (Universal Release)

### 🌟 Overview
An automation tool designed for SUSTech students to simplify the process of bulk downloading resources from the Blackboard (BB) system.

### ✨ Key Features
* **Full Auto-Scanning:** Automatically navigates through chapters, weeks, labs, and assignments.
* **Smart Filtering:** Downloads only document files while skipping messy `.htm` and `.html` links.
* **Always-on-Top GUI:** The control panel floats above your browser for seamless multitasking.
* **Zero Configuration:** Automatically saves to your local `Downloads` folder—works on all Windows PCs.

## 🇫🇷 Français

**Développeur:** Su Xiaowei  
**Version:** v2.4 (Version Universelle)

### 🌟 Aperçu
Un outil d'automatisation conçu pour les étudiants de SUSTech afin de faciliter le téléchargement massif de ressources sur Blackboard.

### ✨ Caractéristiques principales
* **Scan Automatique:** Détecte intelligemment les "Semaines", "Chapitres", "Laboratoires" et "Devoirs".
* **Filtre Intelligent:** Ignore les fichiers `.htm` et ne télécharge que les documents réels (PDF, PPTX, etc.).
* **Interface au Premier Plan:** L'application reste visible au-dessus du navigateur pour une gestion facile.
* **Chemin Universel:** Sauvegarde automatiquement dans le dossier `Téléchargements` de l'utilisateur.
